\section*{Introduction}
\begin{parag}{Overview of the course}
	The course is a 4cts course taken by computer science, communication system, chemistry, environmental science. The course covers the following topics:
    \begin{itemize}
		\item Complex analysis, complex differentiation, complex integration, Laurent series, residue theorem
		\item Review of Fourier transform/series
		\item Laplace transform
		\item Applications to ODES/PDES (initial and Cauchy value problems)
    \end{itemize}
\end{parag}
\begin{parag}{Main textbook}
	The main book that this course refers to is '\textit{Analyse avancée pour ingénieur}' by Dacorrogna Bernard and Tanteri Chiara.
    
\end{parag}
